# Diabetes Dataset EDA

## Overview
This project performs Exploratory Data Analysis (EDA) on the Diabetes dataset to understand patterns and relationships between different health factors.

## Objective
- To analyze the diabetes dataset
- To identify important features like Glucose, BMI, and Age
- To understand how these factors affect diabetes

## Dataset
The dataset contains medical information such as:
- Glucose level
- Blood pressure
- BMI
- Age
- Insulin

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

# Analysis
- Data cleaning and preprocessing
- Handling missing values
- Visualization using graphs (histplot, scatterplot, heatmap)

## Key Insights
- Higher glucose levels are strongly related to diabetes
- BMI and Age also show noticeable impact

## Files
- diabetes.ipynb → Code
- diabetes.csv → Dataset
- diabetes_data_analysis.pdf → Presentation

## Conclusion
This analysis helps in understanding the key factors affecting diabetes and provides useful insights through visualization.
